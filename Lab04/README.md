# List ADT (Singly Linked List)

## Description
Implementation of List Abstract Data Type using singly linked list.

## Operations
1. Insert Beginning
2. Insert End
3. Insert Position
4. Delete Beginning
5. Delete End
6. Delete Position
7. Search
8. Display
9. Display Reverse
10. Reverse Link
11. Exit

## Compilation
```bash
g++ main.cpp -o list_adt_sll
```
